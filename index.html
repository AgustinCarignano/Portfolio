Sobre mí

¡Hola! Mi nombre es Agustín, y estoy estudiando la carrera de Desarrollador Full Stack.
Comencé a estudiar a mediados del 2022, habiendo completado ya los cursos de "Desarrollo Web", "Java Script" y "React JS". Actualmente me encuentro cursando el módulo de "programación Back End", con el que completaré esta formación inicial de Fulls Stack.
Siempre me interesó el mundo de la programación, aunque mi curiosidad por entender como se crean las cosas que usamos de manera cotidiana me decantó por la carrera de Ingeniería Química en su momento. La programación fue primero un pasatiempo, leyendo algo de aquí y allí, hasta que decidí comenzar una formación más seria, la cual sigo trabajando.
Actualmente, busco mi primer experiencia profesional como desarrollador junior, para seguir aprendiendo sobre este apasionante mundo de la programación.

Proyecto Desarrollo Web

Barroca es un sitio web estático de un e-commerce, creado como trabajo final del curso de Desarrollo Web, utilizando HTML, CSS, SASS y Bootstrap.

Proyecto Java Script

Barroca evolucionó mientras desarrollaba el curso de JS, pasando ahora de una web estática a una aplicación que permite al usuario interactuar con el contenido, elegir sus productos y simular un proceso de compra. Apliqué en este proyecto las funcionalidades básicas del lenguaje como ser: funciones, objetos, JSON, AJAX.

Proyecto React JS

Tyrion es un e-commerce de bebidas creado como proyecto para el curso de React JS. En este app web se simula casi al completo el proceso de compra, pudiendo generar una orden que es enviada a una base de datos y que luego puede ser recibida por un hipotetico vendedor. En este trabajo utilicé Create React App, React Router Dom, Firestore de Firebase como base de datos, logrando las funcionalidades mediante AJAX, Context, conditional rendering, etc.
